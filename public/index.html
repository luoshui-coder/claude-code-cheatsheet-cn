📋 最近更新（中文）
  • 新增 --bare：用于脚本化 -p 调用的极简无头模式，不跑 hooks / LSP / 插件扫描
  • 新增 --channels：权限请求可中继到手机端，部分 MCP 消息也可推送
  • 新增 skills / slash commands 的 effort 前言字段
  • /fork 更名为 /branch（旧别名仍兼容）
  • SendMessage 现在可自动恢复已停止的 agent
